# PDF 智能解析助手

使用 Vue + TypeScript + Vite 构建的单页工具，支持：

- 拖拽或选择 PDF 文件并提取文字
- 将提取内容提交给 OpenAI ChatGPT，生成 8 字以内总结和价格
- 在浏览器端重命名 PDF 并下载

## 快速开始

```bash
npm install
npm run dev
```

启动后访问 `http://localhost:5173`。

## 使用说明

1. 在页面中拖入或上传 PDF 文件，等待解析完成。
2. 填写 OpenAI API Key（需要具有所选模型的调用权限）。
3. 点击「生成总结与价格」，等待接口返回。
4. 在重命名区输入新的文件名（会自动补全 `.pdf`），生成并下载即可。

> **提示**：浏览器无法直接覆盖原文件，重命名功能会返回一个新的下载链接，文件内容保持不变但文件名更新。
